# Pilot Documentation

Welcome to **Pilot** — an AI-powered coding assistant built on Electron, React, and the Pi SDK. Pilot helps you write code, manage tasks, maintain context across projects, and accelerate your development workflow with intelligent agent assistance.

---

## What is Pilot?

Pilot is a desktop application that combines:

- **AI-powered chat sessions** for coding assistance with context-aware tools
- **Multi-tab workflow** for working across multiple projects simultaneously
- **Intelligent memory system** that preserves context across sessions (global and project)
- **Integrated task management** with kanban boards, filters, and agent integration
- **Sandboxed tool execution** for safe file operations and bash commands
- **Git integration** with visual diff review and commit workflows
- **Developer tools** including an integrated terminal and extension system

## Quick Reference

### Essential Shortcuts
| Action | Shortcut |
|--------|----------|
| New session | `Cmd+N` |
| New tab | `Cmd+T` |
| Close tab | `Cmd+W` |
| Command palette | `Cmd+K` |
| Tasks board | `Cmd+Shift+T` |
| Steer agent | `Enter` (while streaming) |
| Follow-up | `Alt+Enter` (while streaming) |
| Toggle sidebar | `Cmd+B` |
| Show shortcuts | `Cmd+/` |

See [Keyboard Shortcuts](./keyboard-shortcuts.md) for the complete reference.
