# Keyboard Shortcuts

Pilot is designed for keyboard-driven workflows. This reference covers all keyboard shortcuts, organized by category.

**Quick access**: Press `Cmd+/` anytime to view keyboard shortcuts in the app.

## Customizing Shortcuts

### Keybinding Settings

Customize keyboard shortcuts:
1. Open Settings: `Cmd+,`
2. Go to **Keybindings** tab
3. Click on a shortcut to rebind
4. Press the new key combination
5. Click **Save** or press `Enter`

### Resetting Shortcuts

Reset to default keybindings:
1. Settings → Keybindings tab
2. Click **"Reset to Defaults"**
3. Confirm

**Warning**: This will remove all custom keybindings.

## Platform Differences

### macOS (Default)

All shortcuts above use `Cmd` (Command key).

### Windows & Linux

On Windows and Linux, replace `Cmd` with `Ctrl`:

| macOS | Windows/Linux |
|-------|---------------|
| `Cmd+N` | `Ctrl+N` |
| `Cmd+T` | `Ctrl+T` |
| `Cmd+K` | `Ctrl+K` |
| etc. | etc. |

**Special cases**:
- `Cmd+,` (Settings) → `Ctrl+,`
- `Cmd+Q` (Quit) → `Alt+F4`
- `Cmd+M` (Minimize) → `Windows+Down`

## Tips & Tricks

### Keyboard-First Workflow

1. **Never touch the mouse**:
   - `Cmd+K` → Command palette for everything
   - `Cmd+Tab` → Switch apps (OS-level)
   - `Cmd+1-9` → Jump between tabs

2. **Combine shortcuts**:
   - `Cmd+N` → `Cmd+L` → Start typing → Create session and start chatting immediately
   - `Cmd+Shift+T` → `Cmd+F` → Filter tasks without leaving keyboard

3. **Learn the essentials first**:
   - `Cmd+K` — Command palette (most important)
   - `Cmd+N` / `Cmd+T` — New session/tab
   - `Cmd+B` / `Cmd+Shift+B` — Toggle panels
   - `Cmd+/` — Keyboard shortcuts reference

### Muscle Memory

Practice common workflows:
- **Start a new feature**: `Cmd+N` → type project name → start coding
- **Review changes**: `Cmd+Shift+D` → `Cmd+Enter` → accept changes
- **Check tasks**: `Cmd+Shift+T` → review board → `Esc` → back to chat

### Cheat Sheet Mode

Press `Cmd+/` for a searchable keyboard shortcuts panel:
- Type to filter shortcuts (e.g., "task" shows all task-related shortcuts)
- Click a shortcut to see a demo
- Print or export for reference

## Accessibility

### Screen Reader Support

Pilot supports screen readers with:
- Keyboard navigation for all features
- ARIA labels on interactive elements
- Shortcut announcements

**Recommended**: Enable "Announce shortcuts" in Settings → Accessibility.

### High Contrast Mode

Enable high contrast mode:
1. Settings → Appearance → Theme
2. Select "High Contrast"
3. Shortcuts remain the same

### Reduced Motion

Disable animations:
1. Settings → Accessibility → Reduce Motion
2. Toggle on
3. Keyboard shortcuts are unaffected
